
# Student Performance Analysis using Pandas

## Overview

This project explores a student performance dataset using Python and Pandas. The objective is to perform basic data analysis, answer practical business questions, and strengthen fundamental data analysis skills through real-world data exploration.

The analysis focuses on understanding student study habits, attendance, class participation, grades, and overall performance.

## Dataset

The dataset contains **1,000,000 student records** with the following features:

* student_id
* weekly_self_study_hours
* attendance_percentage
* class_participation
* total_score
* grade

## Objectives

* Number of rows and columns in the dataset
* Student with the highest study hours
* Student with the highest attendance
* Top students based on study hours
* Maximum values across different performance metrics
* General exploration of student performance data

* Reading and exploring datasets with Pandas
